Release notes for LensKit 1.0.2

The Git changelog and the list of closed tickets and pull requests provide more information on what has happened, including bugs that have been fixed.

This release contains bug fixes against 1.0.1.

  • Fix graph pruning so transient dependencies aren’t serialized.

  • Fix transient tags and serializability of item-item recommender models.
