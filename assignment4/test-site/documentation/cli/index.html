The LensKit binary distribution contains a command line tool, lenskit, for performing various LensKit operations. Each operation is exposed as a subcommand, much like git or hg uses, with its own manual page:

  • lenskit (the main tool)
  • lenskit-version
  • lenskit-eval
  • lenskit-grain-model
  • lenskit-predict
  • lenskit-recommend
  • lenskit-graph
  • lenskit-pack-ratings
